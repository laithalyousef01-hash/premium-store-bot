# ملخص المشروع: Telegram Digital Store Bot

## الهدف

إنشاء **بوت متجر على Telegram** لبيع الاشتراكات الرقمية مثل:

* Netflix
* ChatGPT Plus
* PlayStation Plus
* ألعاب
* خدمات رقمية أخرى

بحيث يستطيع العميل:

1. فتح البوت
2. اختيار الاشتراك
3. اختيار الخطة
4. اختيار طريقة الدفع
5. إرسال صورة الدفع
6. وصول الطلب إلى **جروب الدعم**
7. متابعة الطلب يدويًا

---

# ما تم إنجازه

## 1️⃣ إنشاء بوت Telegram

تم إنشاء البوت عبر **BotFather**.

اسم البوت:

```
Premium Subs Jordan
```

الأوامر تم تنظيفها بحيث يبقى فقط:

```
/start
```

وتم حذف باقي الأوامر باستخدام:

```
/setcommands
/empty
```

المصدر:
Telegram BotFather commands documentation
[https://core.telegram.org/bots#botfather](https://core.telegram.org/bots#botfather)

---

# 2️⃣ هيكل المشروع

المشروع مكتوب بلغة **Python** باستخدام مكتبة:

```
python-telegram-bot
```

المصدر:
[https://docs.python-telegram-bot.org](https://docs.python-telegram-bot.org)

هيكل المشروع:

```
telegram_store_bot
│
bot.py
config.py
database.py
products.json
.env
```

---

# 3️⃣ الملفات

## bot.py

الملف الرئيسي للبوت.

المهام:

* عرض القائمة الرئيسية
* عرض الاشتراكات
* عرض الخطط
* اختيار طريقة الدفع
* استقبال صورة الدفع
* إرسال الطلب إلى جروب الدعم
* استقبال طلب لعبة

---

## config.py

يحتوي إعدادات البوت:

```
BOT_TOKEN
SUPPORT_GROUP_ID
STORE_NAME
CLIQ_NAME
USDT_WALLET
USDT_NETWORK
SUPPORT_USERNAME
```

---

## products.json

يحتوي المنتجات والخطط مثل:

```json
{
 "subscriptions": [
  {
   "id": "netflix",
   "name": "Netflix",
   "plans": [
    {"name": "1 Month", "price": 4},
    {"name": "3 Months", "price": 10}
   ]
  }
 ]
}
```

---

## database.py

يستخدم لحفظ الطلبات.

حاليًا يقوم بتوليد:

```
Order ID
```

لكل طلب.

---

## .env

يحتوي معلومات حساسة:

```
BOT_TOKEN
SUPPORT_GROUP_ID
```

ولا يتم وضعها داخل الكود للحماية.

# 7️⃣ طلب الألعاب

إذا اختار العميل:

```

```

```

```

```
EA FC 25 - PS5
```

ويصل الطلب إلى جروب الدعم.

# المشاكل التي واجهناها

سببها:

```
توكن غير صحيح
```

```
تم كشف التوكن
```

```
/revoke
```

[https://core.telegram.org/bots/api#getme](https://core.telegram.org/bots/api#getme)
